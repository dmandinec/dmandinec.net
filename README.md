# dmandinec.net

> My personal website - built with Astro and Three.js

A modern, interactive personal website showcasing my work and thoughts. Features a dynamic 3D background powered by Three.js and a fast, content-focused design with Astro.

## Features

- **Live 3D Background** - Interactive Three.js scene with animated starfield and black hole effects
- **Fast & Lightweight** - Built with Astro for minimal JavaScript and optimal performance
- **Content** - Markdown-based blog-type posts with automatic routing
- **Modern UI** - Clean, responsive design

## Used technologies

- **Framework**: [Astro](https://astro.build/)
- **3D Graphics**: [Three.js](https://threejs.org/)
- **Language**: JavaScript
- **Styling**: CSS
- **Hosting**: [Vercel](https://vercel.com/)

MIT - Feel free to use this as inspiration for your own site!
